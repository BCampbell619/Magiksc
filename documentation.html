The magiksc Template Web Dev Documentation

This page lists all the ‘details’ about the web development of the magiksc.com store.
It has all the layout, code, and codesheets that are associated with each page.
We start with the header and go all the way to the footer!

Store Index Page Files

Header Files

The HTML header file for the store can be found here: store/includes/templates/magiksc/common/html_header.php
In this file is the tag and the <head>, <title>, <meta>, <link>, & <script> tags are found. The Bootstrap CDN and Google Font api is also located in this file. All .css and .js files are loaded in this page.

Body Files

The first body file is found here: store/includes/templates/magiksc/common/tpl_main_page.php
The opening <body> is found in this file, as well as the <div> with the mainWrapper id. There are a few php included files in this document. They are included in the following order:

  1. tpl_header.php
  2. column_left.php
  3. $body_code (This variable is determined in a file found in the file path store/includes/templates/template_default/common/main_template_vars.php)
  4. column_right.php
  5. tpl_footer.php

Each of these will be reviewed futher.

In between column_left.php and $body_code is where the breadcrumb navigations is declared.

tpl_header.php

This file has some base code that displays a default navigation, yet there is a flag that can turn that off. The variable $flag_disable_header is set to true currently for this page. This customer header is wrapped in a <div> with the container–fluid class. The CURRENT TURNAROUND, CONTACT, ABOUT US, the INSTAGRAM links, & the Magik Graphis logo links are all found in this file, as well.

column_left.php

The column_left.php file is found through the following path: store/includes/modules/column_left.php
column_left.php is more of a function type file, which loads through all of the displayable files that have been approved through configuration. Some of these, for example, are the Categories & Information boxes or files . Whatever you'd like to see or not see through this file needs to be turned off or on through the admin portal. Essentially this loads all the sideboxes files for each page. Currently, the only two boxes that are being displayed are the Categories and Information sideboxes. The only file that has custom scripting is the Categories sidebox. This is found through the following file path: store/includes/templates/magiksc/sideboxes/tpl_categories.php. For the magiksc template the Information file is found through the following file path: store/includes/templates/template_default/sideboxes/tpl_information.php.

tpl_categories.php

This file was modified by adding specific id or classes to the <a> tags to apply the template styling needed for the site. lines 13 to 18 is used for troubleshooting or viewing the data structure, and lines 22 to 129 is the block section that sets the styles to the top nav items and each sub–item of the product navigation.

$body_code

This is the main page or file that is loaded to the ‘center’of the store's main page. This main page is found through the following file path: store/includes/languages/english/html_includes/magiksc/define_main_page.php.

column_right.php

This column is completely turned off for this template.

tpl_footer.php

Much like the header file, this file has a flag variable ($flag_disable_footer) set to true. Inserted into this file is the custom code to display the custom footer for the site. This footer file can be found through the following file path: store/includes/templates/magiksc/common/tpl_footer.php.

Categories Page Files

List of Files

Here are the list of files that make up the Categories page display:

  1. tpl_index_categories.php
  2. tpl_modules_category_row.php

The first is found through the following path: store/includes/templates/magiksc/templates/tpl_index_categories.php. The second file is found through the following path: store/includes/templates/magiksc/templates/tpl_modules_category_row.php

tpl_index_categories.php

The returned mark up for this file is encompassed in a row class and a col-12 class within it. This file goes through and loads potentially up to 4 sections of the page – category rows, featured products, specials, & whats new items.

tpl_modules_category_row.php

All this file does is include two additional files that get processed and displayed to the page: category_row.php & tpl_columnar_display.php. The category_row file can be found via: store/includes/templates/module/magiksc/category_row.php, and the other file can be found through : store/includes/templates/magiksc/tpl_columnar_display.php.

category_row.php

This file prepares the format of the output of each of the sub categories for the selected main category (if there are sub categories). If any display changes need to be made it should be through this file.

tpl_columnar_display.php

This file actually takes care of the display of the work done by the category_row.php file. It is in this file that each of the sub category items are wrapped in col-12 col-sm-12 col-md-6 col-lg-4 col-xl-4 classes.

Category List Page Files

List of Files

Here is the list of files that make up the Categories List page display:

  1. tpl_index_product_list.php
  2. tpl_modules_product_listing.php

The first file can be found through this path: store/includes/templates/magiksc/templates/tpl_index_product_list.php. This file is just like the tpl_index_categories.php file in that it loads the same 4 possible modules. The second file can be found through this path: store/includes/templates/magiksc/templates/tpl_modules_product_listing.php. This file requires one document, which is tpl_tabular_display.php.

tpl_tabular_display.php

This file is found via : store/includes/templates/magiksc/common/tpl_tabular_display.php. This file handles all of the parsing and displaying of each of the items found for a given sub category. Any changes to the display or arrangement of the items can be done through this file.

Product Info Page Files

List of Files

Here are the files that make up this page:

  1. tpl_product_info_display.php
  2. tpl_products_next_previous.php
  3. tpl_modules_main_product_image.php
  4. tpl_modules_attribues.php

Here are the file paths for each of these files:

  1. store/includes/templates/magiksc/templates/tpl_product_info_display.php
  2. store/includes/templates/magiksc/templates/tpl_products_next_previous.php
  3. store/includes/templates/magiksc/templates/tpl_modules_main_product_image.php
  4. store/includes/templates/magiksc/templates/tpl_modules_attribues.php

The tpl_product_info_display.php loads the other three files within itself. This file is the master file for this page of the site.
The tpl_products_next_previous.php takes care of the style and mark up of the ‘Previous’, ‘Next’, and ‘Listing’ buttons at the top of the product. Go to this file to modify the look of these.
The tpl_modules_main_product_image.php holds the javascript code that handles the pop out of the main image click. If the behavior of this needs to change, go to this file.
The tpl_modules_attribues.php is the parsing/display file for the attributes of a product. An additional file that's associated with this tpl_modules_attribues.php file is store/includes/modules/magiksc/attribues.php. The attributes.php file is the heavy lifter of parsing all the data that comes from the database regarding attributes. If there are any needed changes for the actual <input> or <label> tags, then this is the file you need.
